24 ####**Project Synopsis**
25 Robotic sensor administration application interface framework. The final goal of this project is to create a [ROS][1]-based system to allow interactive communication between a sensor device and a cloud. It is being developed for [Ubuntu][5] 12.04-15.04, although it should run on other UNIX/Linux-based systems. This package also includes basic simulated TCP/IP message topics for imu (accelerometer), compass, image, and pointcloud types.
26 **Warning: This project is currently in a Beta phase. Expected completion date is May 4 2015.**
27 **Warning: Installer scripts have not yet been fully tested. Since they make modifications on an operating systems level, we do not advice using them at this point.**

41 ####**Installation (Linux/ROS Version)**
42 
43 Follow the instructions for installing [ROS][1]. It is strongly advised to choose
44 the ```ros-<version>-desktop-full``` option to install all necessary ROS packages.
45 *Note: This step is not necessary for running the **frontend** or **prototype**.*
46 
47 Install additional packages via ros:
48 ```
49 sudo apt-get install ros-indigo-sound*
50 sudo apt-get install ros-indigo-pocketsphinx
51 ```
52 
53 Install pocketsphinx dependency:
54 ```
55 sudo apt-get install gstreamer0.10-gconf
56 ```
57 
58 
59 Install the [Boost][4] C++ libraries:
60 ```
61 sudo apt-get install libboost-all-dev
62 ```
63 
64 Install [QT4][3]:
65 ```
66 sudo apt-get install libqt4-*
67 ```
68 
69 Install [QGLViewer][8]:
70 ```
71 sudo apt-get install libqglviewer2 libqglviewer-dev
72 ```
73 
74 Now you must make some modifications so it can be compiled
75 with CMake: ```sudo vim /usr/include/QGLViewer/vec.h```. Now change
76 ```#include <QDomElement>``` to ```#include <QtXml/QDomElement>```.
77 Then save and quit. Next, run ```sudo vim /usr/include/QGLViewer/config.h```.
78 Change ```#include <QGLWidget>``` to ```#include <QtOpenGL/QGLWidget>```.
79 
80 Install [QJson][7]
81 ```
82 sudo apt-get install libqjson0 libqjson-*
83 ```
84 
85 Install [SQLite][6]

87 sudo apt-get install sqlite3 libsqlite3-dev
